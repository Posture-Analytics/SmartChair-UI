# How to run the app

Activate the virtual environment:

```bash
python3 -m venv app
```

Initialize the virtual environment:

```bash
app\Scripts\Activate.ps1 # Windows
```

```bash
source app/bin/activate # Linux
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
python app.py
```
